采集任务管理页面修复

修复内容：

  • ✅ 修复了元素ID不匹配的问题（filterType → typeFilter, filterStatus → statusFilter）
  • ✅ 将卡片列表渲染改为表格行渲染
  • ✅ 添加了事件绑定函数 bindEvents()
  • ✅ 添加了 resetFilters() 函数
  • ✅ 添加了 editTask() 和 updateTask() 函数
  • ✅ 更新了按钮样式类名（action-btn-*）
  • ✅ 添加了空值检查，防止null错误

主要变更：

1. 元素ID修复

// 之前
const filterType = document.getElementById('filterType').value;
const filterStatus = document.getElementById('filterStatus').value;

// 修复后
const typeFilterEl = document.getElementById('typeFilter');
const statusFilterEl = document.getElementById('statusFilter');
const filterType = typeFilterEl ? typeFilterEl.value : '';
const filterStatus = statusFilterEl ? statusFilterEl.value : '';

2. 渲染方式改变

// 之前：渲染卡片
taskList.innerHTML = tasks.map(task => renderTaskCard(task)).join('');

// 修复后：渲染表格行
tbody.innerHTML = tasks.map(task => renderTaskRow(task)).join('');

3. 新增功能

  • 编辑任务功能
  • 更新任务功能
  • 重置筛选功能
  • 事件绑定管理
